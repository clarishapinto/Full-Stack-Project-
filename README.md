# Full-Stack-Project-
Web Application
**Pawisitve Homes** is a full-stack web application developed to promote responsible dog adoption by connecting potential adopters with dogs in need of a loving home through a simple and accessible digital platform.

The application focuses on usability, real-world impact, and clean system design, making the adoption process transparent and easy for users while allowing administrators to manage adoption listings efficiently.

## ğŸ” Environment Variables & Security

Sensitive information such as database credentials is not committed to the repository.  
All confidential values are managed using environment variables.

Create a `.env` file in the root directory with the following configuration:

DB_HOST=localhost  
DB_PORT=5432  
DB_NAME=your_database_name  
DB_USER=your_username  
DB_PASSWORD=your_password
